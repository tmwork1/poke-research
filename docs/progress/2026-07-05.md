# 進捗メモ 2026-07-05

## 今日の作業
- M3 の残タスク「個人ブログ（RSS）」の実現方式を検討した。
  - はてなブログ個別の RSS フィード自体は標準的に配信されており（`https://<blog>.hatenablog.com/rss`、RSS 2.0、`title`/`link`/`description`(本文込み)/`pubDate`/`guid`/`category`）、特定ブログを購読する方式なら技術的には問題なく実装できることを curl で確認した。
  - 一方で「ブログを個別に監視するのではなく、記事を定期的に横断検索したい」という要望が出たため、はてなブックマークの検索 RSS（`b.hatena.ne.jp/q/<keyword>?mode=rss`）を Qiita/Zenn/note と同様のキーワード検索型ソースとして使う案を検討した。
  - 最終的に、ToS が明確で公式ドキュメントのある Brave Search API を使う方針に決定した（はてなブックマーク検索はブックマークされた全ウェブが対象で、かつ非公式寄りの利用になるため）。
  - Brave Search API は API キーが必要な有料サービス（無料枠あり）のため、実装は今回のセッションでは行わず、リリース後のアップデート対応とし、ロードマップにその旨を明記した。M3 の完了条件からは外した。
- 上記決定に伴い、`docs/development-roadmap.md` の M3 セクションを更新し、Zenn/note の拡張が完了した状態で M3 を完了扱いとした。個人ブログ（Brave Search API 経由、リリース後対応）は将来対応として注記のみ残した。
