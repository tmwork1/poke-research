# Pokemon Research

## コンセプト

ポケモンプログラミングに関する技術情報を集約した、日本語圏向けの情報ハブ。

記事・ライブラリ・GitHubリポジトリ・論文・動画などを横断的に検索・閲覧できるプラットフォームを目指す。

単なるリンク集ではなく、AIによる要約・分類や、技術同士の関連性を可視化することで、学習や開発を支援する。

# 開発方針

まずは、ポケモンプログラミングに関する情報を一箇所で探せるプラットフォームを構築する。

記事収集・AI要約・検索機能を中心としたMVPを実装し、その後、ライブラリ・論文・動画・開発者情報などを順次統合する。

将来的には、日本語圏におけるポケモンプログラミングの知識基盤となることを目標とする。
